Dragon Wings

1. What OpenSea Displays (parsed from tokenURI):

Dragon Wings

Type: back

2. Raw SVG Data (stored in gene[tokenId] mapping):

  "description": "A Gene trait of type: back",

  "attributes": [
    {
      "trait_type": "Type",
      "value": "back"
    }
  ]
}
